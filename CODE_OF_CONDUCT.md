# Code of Conduct

This project adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct. The full text — including the standards we hold ourselves to, examples of acceptable and unacceptable behavior, the enforcement ladder, and reporting guidelines — is canonical at that link.

## In short

Everyone is welcome here. Be respectful, assume good faith, focus on what is best for the community, and help newcomers when you can. Behavior that makes others feel unwelcome or unsafe is not tolerated.

## Reporting

If you experience or witness behavior that violates this code of conduct, please report it privately to the project maintainers by emailing `security@price.games` (the same private channel documented in [`SECURITY.md`](./SECURITY.md)). All reports are handled confidentially.

## Enforcement

Maintainers are responsible for clarifying and enforcing this code of conduct. They may take any action they deem appropriate, up to and including a permanent ban from project spaces.

## Scope

This code of conduct applies in all project spaces — the GitHub repository, issue and PR discussions, and any other forum operated under the project's name. It also applies when an individual is officially representing the project in public spaces.
